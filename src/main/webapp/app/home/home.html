Introduction

Meetme is a multi-party video conferencing web application, based on the vidyo.io platform, that allows you to meet with anyone in the world with a single link. This application is open-source and is available for download on GitHub.

Geting Started

Before we get started we need a little bit of information to configure your first meetme application:

  1. Grab an Application ID and Developer Key from your API Key's. They are used to create tokens automatically and will always remain hidden.
  2. Create a Application with an Application ID (domain) and Developer Key from vidyo.io.
  3. Create a Meeting for a given Application.
  4. Try out the meeting link by clicking the Launch button.
  5. Invite others with the meeting link you just created.

Notes

  • Your vidyo.io Application ID and Developer Key are kept private at all times.
  • The links are are not guessable so no one can use your minutes without knowing the specific meeting link. Delete meeting links once a meeting is over for the greatest security.
  • If you need more features than those available, this application is open-source and you can modify it as you need on GitHub.
